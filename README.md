# 🔒 LanChat — Secure LAN Messenger

![C++](https://img.shields.io/badge/C%2B%2B-23-blue.svg)
![Qt](https://img.shields.io/badge/Qt-6.0%2B-green.svg)
![License](https://img.shields.io/badge/License-MIT-lightgrey.svg)

**LanChat** — це захищений консольний месенджер для локальних мереж, розроблений в рамках курсової роботи. Проєкт реалізує клієнт-серверну архітектуру з використанням TCP-сокетів та забезпечує конфіденційність листування завдяки симетричному протоколу шифрування.

## 🚀 Особливості

* **Архітектура Клієнт-Сервер:** Сервер виступає ретранслятором повідомлень, підтримуючи багатокористувацький режим.
* **End-to-End Шифрування:** Використовується симетричне XOR-шифрування (гаммування). Сервер не може прочитати зміст повідомлень.
* **JSON Протокол:** Дані передаються у структурованому форматі JSON.
* **CLI Інтерфейс:** Клієнт підтримує аргументи командного рядка для швидкого підключення (IP, порт, нікнейм, ключ).
* **Кросплатформеність:** Завдяки Qt 6 код працює на Windows, Linux та macOS.

## 🛠️ Технологічний стек

* **Мова:** C++23
* **Фреймворк:** Qt 6 (Network, Core)
* **Збірка:** CMake 4.1+
* **Формат даних:** JSON

## 📂 Структура проєкту

Проєкт розділено на модулі для забезпечення чистоти архітектури:

* `Common/` — Спільна статична бібліотека.
* `Server/` — Серверний додаток (керування сокетами, бродкастинг).
* `Client/` — Клієнтський додаток (авторизація, парсинг аргументів).
